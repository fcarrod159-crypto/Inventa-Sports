# Inventa-Sports
Se trata de una Situación de Aprendizaje para el área de Educación Física y destinada a alumnado de tercer ciclo de primaria. En ella tendrán que desplegar sus competencias para crear su propio deporte y utilizar los recursos digitales. 
Es un recurso educativo creado com exelearning, puedes ver el recurso funcionando on line en el siguiente enlace: https://fcarrod159-crypto.github.io/Inventa-Sports/
También puedes descargarte el proyecto editable, descargando el archivo .elp que aparece en la lista de archivos o en el apartado de créditos.
